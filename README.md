# LALS

LALSは赤外線射的ゲーム用のライブラリ群です。

LALSという名称は`LALS: Are Libraries for Syateki`の略から来ています。

## インストール方法

ソースファイル一式をダウンロードして、自分のプロジェクトにコピーしてください。

## 対象ハード

以下を推奨します。

- M5StickC
- M5StickC Plus
- M5Stack

## 使用例

`LALS.ino`を参照してください。

- [ ] まと毎の赤外線受光時処理
- [ ] まと毎の赤外線非受光時処理
